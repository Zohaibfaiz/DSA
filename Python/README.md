# Python Learning Notes

This workspace contains chapter-wise Python practice files and a small FastAPI example.

## What is in this project

- `chp1.py` covers Python basics such as printing, variables, data types, type conversion, input handling, arithmetic, comparison operators, and simple calculations like sum, square area, and average.
- `chp2.py` focuses on strings and conditionals. It includes string creation, indexing, slicing, common string methods, if/elif/else logic, even/odd checks, grading logic, max-of-three checks, and multiples of 7.
- `chp3.py` covers lists and tuples. It includes list indexing and slicing, list methods like `append`, `sort`, `reverse`, `insert`, `remove`, and `pop`, tuple operations, list-based mini programs, palindrome checks, searching in lists, password length checks, and bill calculation with discount logic.
- `chp4.py` covers dictionaries and sets. It includes nested dictionaries, keys/values/items, `get`, `update`, set creation, set methods, union and intersection, and conversion between list and set.
- `chp5.py` focuses on loops and recursion. It includes `while` loop examples, counting loops, number ranges, multiplication tables, searching in a tuple/list, `continue`, and a recursive factorial attempt.
- `chp6.py` contains file handling and loop practice. It shows an example of writing to a file, counting even numbers in a list, checking string lengths in a list, and a `while` loop solution for filtering names by length.
- `main.py` is a small FastAPI app with routes for a welcome message, a hello endpoint, and a square calculator endpoint.
- `python basics.ipynb` is a notebook version of the same Python practice material, organized into many code cells and one markdown cell.
- `requirements.txt` lists the packages needed for the FastAPI example and related tooling.

## Topics covered

- Python syntax and first program
- Variables, data types, and type conversion
- Input and output
- Operators and comparisons
- Strings and string methods
- Conditional statements
- Lists, tuples, dictionaries, and sets
- Looping with `while`
- Basic recursion
- File handling
- FastAPI routes and simple API responses
