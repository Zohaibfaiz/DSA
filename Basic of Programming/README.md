# Basic of Programming

This folder contains [dsa.ipynb](dsa.ipynb), a Python practice notebook for basic programming and problem-solving concepts.

## What the notebook covers

- Functions and simple reusable logic
- Recursion and recursive thinking
- Loops and conditional statements
- Strings and palindrome checks
- Lists and list operations
- Dictionaries and frequency counting
- Number problems such as Armstrong numbers and divisors
- Sequence generation such as Fibonacci

## What I learned from the notebook

- How to write and call functions for repeated tasks
- How recursion works by solving a problem using smaller versions of the same problem
- How to use loops to process numbers and collections
- How to work with lists, dictionaries, and strings in Python
- How to solve common beginner DSA-style exercises step by step

## Example practice topics

- Check whether a number or string is a palindrome
- Test whether a number is an Armstrong number
- Find factors and divisors of a number
- Count repeated values using a dictionary
- Generate Fibonacci numbers
- Reverse or rearrange strings

## How to use

Open [dsa.ipynb](dsa.ipynb) in VS Code or Jupyter Notebook and run the cells one by one.

If you want, you can extend this notebook by adding more topics such as arrays, sorting, searching, and basic recursion exercises.
